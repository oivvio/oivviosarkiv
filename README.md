Oivvios arkiv
=============

Migrerade oivviosarkiv.polite.se till Pelican i december 2015.

Temat 'martin-pelican-theme' har jag checkat out från github och inkluderat i
projektet med diverse små modifikationer.

Jag har skapat en virtualenv som heter 'oivviosarkiv'. I den kan köra

    fab clean; fab build

för att bygga siten.
